# 所有标记一览

| 默认[^1] | 标记 | 描述 | 备注 |
| --- | --- | --- | --- |
|   | `:d0` | 不显示分割线 | |
| * | `:d1` | 显示默认分割线 (细条样式) | |
|   | `:d2` | 显示更明显的分割线 (粗块样式) | |
|   | `:e0` | 不对错误使用特定样式 | |
|   | `:e1` | 打印错误信息 (红色字体) | |
| * | `:e2` | 优美打印错误堆栈信息 | Rich 库风格 |
|   | `:e3` | 打印错误堆栈以及当前变量字典 (show_locals) | |
|   | `:e4` | ... | 未实现; 可能移除 |
|   | `:f0` | 不启用该功能 | |
| * | `:f1` | 立即输出缓存内容 | |
|   | `:f2` | 立即清空缓存 (但不输出), 并打印当前内容 | |
|   | `:f3` | 立即打印当前内容, 稍后继续处理缓存内容 | 不建议使用 |
|   | `:i0` | 重置计数器 | 仅重置当前作用域, <br>如果在全局使用, 则重置所有作用域的计数器 |
|   | `:i1` | 对当前行计数 | |
| * | `:i2` | 对当前作用域计数 | |
|   | `:i3` | 对全局计数 | |
|   | `:l0` | 不展开对象 | |
| * | `:l1` | 尝试展开对象 | |
|   | `:l2` | 使用特定格式展开对象 | 例如, 嵌套列表以表格形式打印 |
|   | `:n0` | 不显示变量名 | |
| * | `:n1` | 显示变量名 | |
|   | `:p0` | 源信息指向当前帧 | |
| * | `:p1` | 源信息指向上一帧 | |
|   | `:p2` | 源信息指向上上帧 | |
|   | `:p3` | 源信息指向上上上帧 | |
|   | `:p4`+ | ... 上溯更多层级 | 谨慎使用 `:p2` 及以上标记, 可能出现索引溢出错误 |
|   | `:r0` | 不对 Rich 对象使用特定格式 | |
| * | `:r1` | 对 Rich 对象使用特定格式 | |
|   | `:r2` | 同 `:l2` | |
|   | `:s0` | 不裁剪信息 | |
| * | `:s1` | 不显示变量名 | 用于抑制默认配置启用变量名的情况 |
|   | `:s2` | 不显示源信息 | |
|   | `:s3` | 仅使用 Python 内建打印函数 | |
|   | `:t0` | 重置计时器 | 仅重置当前作用域, <br>如果在全局使用, 则重置所有作用域的计时器 |
|   | `:t1` | 对当前行更新计时 | |
| * | `:t2` | 对当前作用域更新计时 | |
|   | `:t3` | 对全局更新计时 | |
|   | `:v0` | 不使用打印级别 | 默认颜色 (黑或白) |
| * | `:v1` | DEBUG 打印级别 | 浅灰色 |
|   | `:v2` | INFO 打印级别 | 蓝色 |
|   | `:v3` | WEAK SUCCESS 打印级别 | 浅绿色 |
|   | `:v4` | SUCCESS 打印级别 | 绿色 |
|   | `:v5` | WEAK WARNING 打印级别 | 浅黄色 |
|   | `:v6` | WARNING 打印级别 | 黄色 |
|   | `:v7` | WEAK ERROR 打印级别 | 浅红色 |
|   | `:v8` | ERROR 打印级别 | 红色 |
|   | `:v9` | CRITICAL 打印级别 | 红底白字. 未实现; 不建议使用; 可能移除 |

[^1]: 星号 (*) 表示默认值.
